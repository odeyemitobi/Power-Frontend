# Power-Frontend

Welcome to the **Power-Frontend** repository! This is the frontend application for the power distribution payment system, built with React, TypeScript, and Vite.

## 🚀 Getting Started

To get started quickly with the codebase, follow these steps:

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/odeyemitobi/Power-Frontend.git
   cd Power-Frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Your application should now be running locally on the port provided by Vite (usually `http://localhost:5173`).

## 🛠 Tech Stack

- **Framework**: React
- **Language**: TypeScript
- **Build Tool**: Vite

## 🤝 How to Contribute

We have a detailed list of feature requests and bugs located in the **Issues** tab of this repository.

1. **Find an Issue**: Browse the open issues and find one you'd like to work on. Each issue contains a detailed description, proposed changes, and a list of files you'll likely need to edit.
2. **Claim the Issue**: Leave a comment on the issue so others know you are working on it.
3. **Branch Out**: Create a new branch for your feature or fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Develop**: Write your code. Make sure to follow the acceptance criteria listed in the issue.
5. **Submit a PR**: Once your work is ready and tested, push your branch and open a Pull Request against the `main` branch. 

## 📝 Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the app for production.
- `npm run lint`: Runs ESLint to catch linting errors.
- `npm run preview`: Locally preview the production build.
